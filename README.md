# Voice Assistant Server

Telegram-бот для голосового ассистента (текстовая версия без микрофона).

## Команды

- погода
- курс доллар / курс евро
- таймер 5 минут
- расскажи анекдот
- сколько время
- открой вк / открой ютуб
- запусти steam / запусти браузер
- сердце 5 красное
- Яндекс погода / Яндекс новости
- какой сегодня день

## Установка и запуск

1. Клонируй репозиторий
2. Создай виртуальное окружение
3. Установи зависимости: `pip install -r requirements.txt`
4. Создай `secrets.py` с токенами
5. Запусти: `python main.py`
