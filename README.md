# Voice Assistant Server

Telegram-бот для голосового ассистента (текстовая версия без микрофона + Docker).

## 🤖 Живой пример

Попробовать бота в деле можно прямо сейчас:  
👉 [**@oleg_assistants_bot**](https://t.me/oleg_assistants_bot)

— Просто напиши ему «погода», «курс доллар» или любой вопрос.*

---
## 📝 Команды

| Тип | Команды |
|-----|---------|
| **Погода** | `погода` |
| **Курс валют** | `курс доллар`, `курс евро` |
| **Таймер** | `таймер 5 минут`, `таймер 1 час`, `таймер 30 секунд` |
| **Анекдоты** | `расскажи анекдот` |
| **Время и дата** | `сколько время`, `какой сегодня день` |
| **Сердце** | `сердце 3` (ASCII-арт) |
| **Расчёт материалов** | `рассчитай стяжку площадь 25 слой 5` |
| **Умный дом** | `включи свет`, `выключи свет` (требует `/auth`) |
| **Общение с ИИ** | Любое сообщение, не похожее на команду, отправится в DeepSeek |

---
## 🔒 Умный дом (авторизация)
Команды включи / выключи требуют авторизации:
1. Отправь боту /auth
2. Введи пароль из SMART_HOME_PASSWORD
3. После этого можно управлять устройствами
---

## 🤖 ИИ-общение
Если отправить боту сообщение, которое не начинается с известной команды (погода, курс, таймер и т.д.), оно будет отправлено в DeepSeek API.
Бот ответит сгенерированным текстом на русском языке.

## 🛠️ Установка и запуск

1. Клонируй репозиторий
2. Создай виртуальное окружение
3. Установи зависимости: `pip install -r requirements.txt`
4. Создай `secrets.py` с токенами
```python
TELEGRAM_TOKEN = "токен_бота"
SMART_HOME_PASSWORD = "пароль_для_умного_дома"
DEEPSEEK_API_KEY = "ключ_deepseek"
```
5. Запусти: `python main.py`
---

## 🐳 Запуск через Docker
1. Создай файл `secrets.py` с токенами (как описано выше)
2. Собери образ:
```bash
docker build -t oleg-telegram-bot .
```
3. Запусти контейнер:
```bash
docker run -d --name oleg-bot --restart always oleg-telegram-bot
```
